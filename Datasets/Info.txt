In this folder are different datasets provided which can be used in the Model.py file.
Other dataset files can also be added and used in the same way.
The only difference is that extra data processing features have been added for these predefined datasets,
what these features are can be found in the overview below.

Overview:
1. Lord_of_the_Rings (Contains .txt files of all three books)
    - Source: https://www.kaggle.com/ashishsinhaiitr/lord-of-the-rings-text
    - Features: Character capitalization (Not created yet)
